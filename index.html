About

I am a Research Engineer at École Centrale de Lyon, currently working on the DECALIGNE project under the supervision of Romain Vuillemot. I received my bachelor's degree in Economics in 2019. I then obtained a Master's degree in Computer Science from Visual Computing Research Center(VCC), where I was supervised by Pengfei Xu. My research interests are in computer graphics, particularly geometry processing, and interactive systems for graphics design and visualization.

Publications

Interactive Element Packing for Animation

Mu Liang, Pengfei Xu*, Hui Huang

2025

PDF(coming soon) · Video · Code

We extend interactive element packing from static layouts to animation generation. A brush-based system allows designers to directly control element motion within a packing pattern, enabling efficient creation of personalized element packing animations.

Brush-based Interactive Element Packing

Mu Liang, Pengfei Xu*, Hui Huang

Journal of Graphics 2024

PDF · Video · Code

Element packing is a visual art form widely used in logos, posters, and advertisements. We propose a brush-based interactive element packing technique that enables designers to perform boundary outlining and element packing simultaneously in real time using a physics-based simulation.

Projects

DeformationQt

DeformationQt is an interactive visualization system for geometric deformation algorithms implemented with a Qt-based graphical interface. The system allows users to explore and compare deformation behaviors in real time through direct manipulation.

Project page

2D Barycentric Coordinates

This project presents a comparative visualization of barycentric coordinate methods in two dimensions, including Wachspress Coordinates, Mean Value Coordinates, Green Coordinates, and Iterative Coordinates.
